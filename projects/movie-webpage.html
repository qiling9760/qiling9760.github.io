The Movie Webpage Project

Context

For my SEP Movie Project, I needed to work with a partner and used our knowledge for basic HTML and CSS to create a fake movie webpage. My partner and I decided to create a horror movie, and the title of it was "Hunter".

Content

To start our project, we needed to make a plan. We plan out our plot for the movie, and what we should include in the webpage in order to reach the requirements. For the HTML part of the movie, we needed to include

- headings - paragraph - divs and spans - imported Google fonts - inputs and submit bottom

For the CSS part, the requirements were

- element and class and id selectors - border and padding and margin properties

There should also be 2 local images that were downloaded and get used in the webpage.

Reflection

Challenge

My partner and I put our best effort into this project to make it as realistic as possible. For us to do that, we searched up additional inforamation online in order to design our webpage. One of it would be adding strokes to the words. I looked up for any CSS properties that could add strokes to words. However, it did not work when I copied the codes into my work. I went to other websites and tried each code they presented, I was finally able to find the correct code and change the strokes of my h1 and h2 elements.

Takeaway

Because I need to add 2 images to my webpage, I decided to use these 2 images as the background of my webpage. I looked online and found a code I can use. 


                background-image: url("img/hunter-hand.jpeg");
