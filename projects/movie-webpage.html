The Movie Webpage Project

Context

For my SEP Movie Project, I needed to work with a partner and use our knowledge of basic HTML and CSS to create a fake movie webpage. My partner and I decided to make a horror movie, and the title was "Hunter."

Content

To start our project, we needed to make a plan. We plan our plot for the movie and what we should include in the webpage to meet the requirements. For the HTML part of the movie, we needed to include

  • headings
  • paragraph
  • divs and spans
  • imported Google fonts
  • inputs and submit button

    • For the CSS part, the requirements were

  • element and class and id selectors
  • border and padding and margin properties

    • There should also be two local images on the webpage.

    Reflection

    Challenge

    My partner and I put our best effort into this project to make it as realistic as possible. To do that, we searched for additional information online to design our webpage. One of them would be adding strokes to the words. I looked up any CSS properties that could add strokes to words. However, it did not work when I copied the codes into my work. I went to other websites and tried each code they presented, and I was finally able to find the correct code and change the strokes of my h1 and h2 elements.

    Takeaway

    I used these two images as the background because I needed to add two images to my webpage. I looked online and found a code I can use.

    
                background-image: url("img/hunter-hand.jpeg");

    Next Step

    I will add a sequel to my movie webpage if I have more time. I already thought up the plot of the next movie with my partner and dropped it down in the planning section, but we did not have enough time to make another html webpage.

    Skills I learned

    This project taught me that I need to try on my own and even search on Google before asking people for help. When I have a problem, I must work it out and try anything I can think of first. I also learned the importance of partnership. I cannot do all the work myself and need to split up the work. When I am stuck, I can also ask her for help. In this way, we are both learning from each other.
